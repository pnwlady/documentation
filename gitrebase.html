Git Rebase

  1. After your final commit on your branch, checkout to master and pull down any possible remote changes to your local master branch
  2. git checkout master
  3. git pull
  4. Now go back to your branch to rebase (CONNECT-549 –feature-blah)
  5. git checkout CONNECT-549 –feature-blah
  6. git rebase –i master(i=interactive)
  7. Work
  8. Git add, commit. When ready to push, checkout to master and pull any updates, check back out to your branch and rebase.
  9. Submit pull request.
  10. Have team member review request, approve and merge. (Reviewer/approver and merger are not the same person.)
  11. Everyone pull down updated master.
  12. Delete local branch.
