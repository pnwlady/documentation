Starting a new project

  1. Bring up Automator and search for MountWarpships.app file. Hit Run.
  2. If that doesn't work try running individual commands in terminal.
  3. mount -shows which servers are mounted
  4. --- warpships scripts ---- mounting scripts
  5. umount ~/mnt/* &>/dev/null
  6. Work
  7. /usr/local/bin/sshfs coal:/home/www/connect/public_html/module/terra/template/default/controller/ ~/mnt/coal/ -o defer_permissions,reconnect,volname=coal
  8. #/usr/local/bin/sshfs spidercliff:/home/www/terrastaffinggroup.com/ ~/mnt/spidercliff/ -o defer_permissions,reconnect,volname=spidercliff
  9. /usr/local/bin/sshfs spidercliff:/home/www/terrastaffinggroup.com/myaccount/dev_html/ ~/mnt/myaccount/ -o defer_permissions,reconnect,volname=myaccount
  10. Everyone pull down updated master.
  11. Delete local branch.
